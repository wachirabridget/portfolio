Project 2: Mobile App

A mobile app designed to help users track their fitness and nutrion goals using React Native.

Project Overview

This mobile app was built using React Native, allowing users to set and track their fitness goals, such as steps, calories burned, and workouts. The app provides a personalized experience and a simple user interface for maximum engagement.

Features:

  • User profile creation and login
  • Track steps, calories, and workouts
  • Push notifications for reminders
  • Integration with device sensors for accurate tracking

Technologies Used:

  • React Native
  • Firebase (for user authentication and data storage)
  • Expo (for development and testing)
  • Redux (for state management)
