Project 1: Dynamic Web Application

A dynamic web application built using HTML, CSS, and JavaScript, designed to deliver a seamless user experience.

Project Overview

This project is a web application that allows users to track and manage savings. The application was built with a focus on simplicity, interactivity, and ease of use. It features a modern, responsive design that works well on all screen sizes.

Features:

  • User authentication (login/signup)
  • Task management with CRUD operations
  • Real-time updates using JavaScript
  • Responsive design with Bootstrap

Technologies Used:

  • HTML5
  • CSS3 (Bootstrap 5)
  • JavaScript (Vanilla JS)
  • Node.js (for backend)
  • MongoDB (for database)
