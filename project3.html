Project 3: E-commerce Site

An e-commerce website built with Django, integrated with payment gateways and a dynamic product catalog.

Project Overview

This project is a fully functional e-commerce website where users can pay fees to various institutions . The site is built with Django and uses PostgreSQL for database management.

Features:

  • Dynamic payment catalog with search and filtering
  • User authentication and order management
  • Payment gateway integration (Stripe)
  • payment tracking and email notifications

Technologies Used:

  • Django (for back-end development)
  • PostgreSQL (for database management)
  • Stripe (for payment integration)
  • HTML5, CSS3 (for front-end design)
  • JavaScript (for dynamic interactivity)
